Frequently Asked Questions:

Q: How do I open the ClickGUI?

A: The default ClickGUI keybind is P. To change this, right click the ClickGUI module and set a bind, or you can use the bind command: .bind clickgui

Q: How do I use commands?

A: The default command prefix for chat is asterisk (.). To use chat commands, you must type this prefix before any command you want to type. For a list of commands, type .help in the game chat.

Q: How do I install Temple Client 1.12.2?

A: To install Temple Client 1.12.2, download any version from the homepage or our GitHub releases page.

Then make sure you have 1.12.2 Forge installed and place the templeclient-version.jar into your mods folder.

Q: The game is crashing, how do I fix this?

A: Please send a crash log to contact@templeenterprise.com for assistance.
